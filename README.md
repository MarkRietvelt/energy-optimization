# Information needs of embedded systems software developers when optimizing for energy consumption
This repository contains the materials, data and analysis for the master thesis titled: "Empowering embedded systems developers to reduce the energy consumption of their software through visualization: uncovering their information needs".

## Materials
The folder "Materials" contains:
- The protocol for the participant involvement sessions
- The questionnaire (available as Qualtrics file and PDF)
- Privacy information
    - Information sheet
    - Consent form
    - Privacy and Ethics Quick Scan

## Data & Analysis
The folder "Analysis" contains the data, and analysis of the reenactment and interview as well as the questionnaire.

### Reenactment & Interview
The transcribed and anonymized recording of the reenactment and interview were summarized for each participant seperately and coded using NVivo 14.23.3.

### Questionnaire
The questionnaire responses were preprocessed and saved into a csv file and analyzed collectively using R 4.4.0 and also saved individually as pdf files. Running the analysis.R file should recreate the graphs in the "Images" folder.
